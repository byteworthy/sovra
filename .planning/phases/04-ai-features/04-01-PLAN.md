<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP client -> MCP server | Tool call arguments cross from Next.js to Go worker |
| Go tool -> filesystem | File tools access container filesystem |
| Go tool -> external HTTP | Web fetch accesses arbitrary URLs |
| Go tool -> PostgreSQL | Database tools query vector_documents |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | file tools | mitigate | filepath.Clean() + HasPrefix check against workspace root; reject traversal |
| T-04-02 | Information Disclosure | web_fetch | mitigate | Blocklist private IP ranges (127/8, 10/8, 172.16/12, 192.168/16) via net.LookupHost before fetch |
| T-04-03 | Information Disclosure | semantic_search | mitigate | Require tenant_id in every query; return error if empty; RLS as defense-in-depth |
| T-04-04 | Denial of Service | all tools | mitigate | context.WithTimeout(30s) on every tool handler |
| T-04-05 | Denial of Service | web_fetch | mitigate | io.LimitReader(1MB) on response body |
| T-04-06 | Information Disclosure | web_search | accept | Brave API key in env var, not logged; no PII in search queries (agent-generated) |
| T-04-07 | Spoofing | MCP server | accept | Internal Docker network only; no public exposure; no auth on MCP endpoint |
</threat_model>
