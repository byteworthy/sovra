Quickstart in minutes

Use the bootstrap script, start local Supabase, then run the web app.

git clone https://github.com/ByteWorthyLLC/sovra.git
cd sovra
pnpm setup:local:start

Full setup and production onboarding docs live in the repository README and docs folder.

Yes. Sovra is self hosted and supports multiple cloud providers.
